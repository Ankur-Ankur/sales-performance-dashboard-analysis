# Sales Performance Dashboard Analysis

## Project Overview

This project analyzes retail sales data using SQL (Google BigQuery) and Tableau to identify key business insights. The dashboard provides an interactive view of sales performance, profitability and regional trends.

---

## Objectives

- Analyze sales performance
- Identify profitable product categories
- Compare regional sales and profits
- Track monthly sales trends
- Identify top-selling and loss-making products
- Build an interactive Tableau dashboard

---

## Tools Used

- Google BigQuery
- SQL
- Tableau Public
- Microsoft Excel
- GitHub

---

## Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Sales vs Profit by Region
- Profit by Category
- Top 10 Products by Sales
- Top 10 Loss-Making Products
- Interactive Filters

---

## SQL Analysis

The project includes SQL queries for:

- Data Exploration
- Data Cleaning
- KPI Analysis
- Monthly Sales Analysis
- Regional Performance
- Product Analysis
- Profitability Analysis

---

## Key Business Insights

- Identified the highest-performing sales regions.
- Compared profitability across product categories.
- Detected products generating consistent losses.
- Analyzed monthly sales trends to identify seasonal patterns.
- Highlighted the top-performing products by revenue.
